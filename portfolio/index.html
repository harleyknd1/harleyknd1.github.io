Project Pheonix

Project Pheonix is a fan project where we reverse engineer the game "APB:Reloaded" and make our own version in Unreal Engine 4. Since this is a massive undertaking its fun and interesting to see how developers back in 2010 worked on the older platforms and develop new methods to do the things they did.

  • Client: None
  • Date: October 2016
  • Service: Game Development

Pong With Lasers
(and rockets)

Pong With Lasers (and rockets) is one of my pet side projects for IGAD (International Game Architecture and Design).
To enter this University of Applied Sciences in Breda.

The game contains the basic elements of pong but is still worked upon.

made with:
• C++
• C#
• Unreal Engine 4.
• And Love

This has become one of my most interesting projects to have worked on.

  • Date: November 2016
  • Service: Game Development
